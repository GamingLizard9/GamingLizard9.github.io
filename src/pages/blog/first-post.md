Hello there, and welcome to my new blog!

I am going to write down my thoughts from time to time, and share interesting or fun things I find.

Don't expect there to be much of a theme; I have a lot of different interests and I want to write about all of them lol. Eventually I'll have tags so you can filter posts to just the ones you want to read, but for now, hopefully this is not too much trouble.

This is still in a very early state. Currently, I get Markdown text formatting, and the ability to attach an image and link to each post. There are some features (tags, RSS, etc.) that I want to add but haven't yet. Hopefully I will have those on here soon.
